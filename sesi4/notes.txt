Tipe Data di JS
- integer (angka bulat, numerik bulat seperti 1 2 3 12093782 423 54)
- float (angka dengan koma, numerik, 1.0 5.5 8.7666 2342421342.435 43.234)
- string (text alfabet+numerik+simbol yang dikutip "HAcktiv8!!!!" 'INPG-pkj-08~~~~~' `halo`)
- boolean (True False) semua hal, semua data di komputer diberikan label True atau False
    komparator > < >= <= == !=
    tipe data lain 0 0.0 "" '' `` FALSE yang lain TRUE

- array
    tipe data yang dibuat menggunakan []
    di dalam [] bisa ada banyak data lainnya, setiap item dipisah dengan ,
    buah_buahan = ["apel", "pisang", "jeruk"] --> array isi tiga string
    untuk mengambil item di dalam sebuah array kita gunakan angka index, dimulai dari 0
    misal mau ambil "apel"
        buah_buahan[0] --> akan dapat "apel"

- object
    tipe data dibuat menggunakan {}
    di dalam {} bisa ada banyak data lainnya, setiap item dipisah dengan ,
    setiap item ada dalam bentuk sepasang key:value
    key selalu string
    value tipe datanya bebas
    buah_buahan = {"apel":5000, "pisang":3000, "jeruk":2500} --> object isi tiga integer
    untuk mengambil item di dalam sebuah object, kita gunakan key nya
    misal mau ambil harga buah apel
        buah_buahan["apel"] --> akan dapat 5000
