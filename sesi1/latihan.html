tag <html>

  • <html> </html>
  • <head> </head>
  • <title> </title>
  • <body> </body>
  • <h1> </h1> untuk judul, text besar dan bold
  • <p> </p> untuk paragraf
  • <img> untuk gambar
  • <br> untuk baris baru
  • <div> </div> untuk apapun, bebas, bisa isi tag lain, bisa isi text
  • <span> </span> mirip div tapi tidak ngambil tempat 1 baris penuh seperti div
  • <ul> </ul> unordered list, untuk bikin list pakai bulat bulat
  • <ol> </ol> ordered list, untuk bikin list yang angka urut 1,2,3,...
  • <li> </li> list item untuk satu barang di dalam ul atau ol
  • <a> </a> untuk link ke website lain
